## Expert adaptation checklist

1. Confirm the sample uses the same source event and entity type as the intended rule.
2. Replace tenant-specific IDs, names, states, roles, fields, URLs, and recipients.
3. Add unchanged-value guards, retry deduplication, and direct/transitive cycle prevention.
4. Validate null, deleted-relation, multi-team, permission, and high-volume behavior.
5. Test in a non-production scope, inspect logs, then document ownership and rollback.
